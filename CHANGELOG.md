## [1.0.0] - 2026-06-22

### Added

- Initial public release.
- Flatten a repository's AI-source into one ordered, token-budgeted context bundle with a table of contents.
- Zero runtime dependencies; requires Node >= 18.
- Hermetic self-test (`npm test`) and CI across Node 18, 20 and 22.
